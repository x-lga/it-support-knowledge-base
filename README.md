# it-support-knowledge-base

Structured knowledge base of 40+ IT support issues with step-by-step resolutions, ITIL 4 priority framing, and escalation criteria. Organised by category: Hardware, Network, Software, Security, Azure, and Identity.

---

## Structure

| Category | Articles | Skills Covered |
|---------|---------|---------------|
| Hardware | 3 | A+ hardware troubleshooting |
| Network | 3 | Network+ TCP/IP, VPN, mapping |
| Software | 3 | A+ OS and application support |
| Security | 3 | Security+ incident response |
| Azure | 3 | AZ-104 cloud support |
| Identity | 3 | AD/Entra ID access management |

---

## Article Format

Every article follows a consistent format:
- **KB number and title**
- ITIL 4 priority classification (P1–P4)
- Average resolution time
- Symptoms
- Step-by-step resolution
- Escalation criteria
- ITIL 4 documentation notes (Incident/Problem/Change classification)

---

## Skills Demonstrated

- Technical breadth across hardware, network, security, cloud, and identity domains
- ITIL 4 applied: every issue is classified by incident type, priority, and escalation threshold
- Documentation discipline: consistent, reusable, SLA-aware article format
- L1 self-sufficiency: each article defines exactly what L1 resolves vs. what escalates

---

## Outcome

A living knowledge base that grows with experience. This format mirrors the knowledge management systems used in real IT support environments (ServiceNow KB, Confluence, Freshservice). The ITIL 4 framing demonstrates that support knowledge isn't just technical - it's process-aware.
